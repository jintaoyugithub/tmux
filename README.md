# Tmux settings and shortcuts

## Shortcuts

- tns <name>

```bash
tmux new-session -s <name>
```

- tas <name>

```bash
tmux attach-session -t <name>
```

- tl

```bash
tmux ls
```

- ts <name>

```bash
tmux switch -t <name>
```

- tkS

```bash
tmux kill-server
```

- tks <name>

```bash
tmux kill-session -t <name>
```

- tkw 

```bash
tmux kill-window
```

- tlw

```bash
tmux list-windows
```
